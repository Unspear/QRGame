Sprites

createSprite(char, color, x, y)

Creates a new sprite and adds it to the world

  • char - A string of characters which will be the appearance of the sprite
  • color - The color id of the sprite, 0 to 7 is a normal color and 8 to 15 is inverted
  • x and y
    • - The position of the sprite in world pixels

destroySprite(target)

Destroys a sprite, removing it from the world. Afterward, the sprite object will still exist in memory and its parameters can be accessed or it could be used as a source for copySprite(...)

  • target - The sprite that will be destroyed

copySprite(source)

Creates a new sprite with the same parameters as source

  • source - The sprite that will be copied
